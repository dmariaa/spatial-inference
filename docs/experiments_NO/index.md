# NO Experiments

This section collects the generated documentation artifacts for the METRAQ NO experiment set.

## Summary Tables

- [Performance Summary](summary_tables/summary_paper_performance.md): paper-oriented aggregate metrics, win rates, and pairwise Wilcoxon outcomes for baseline, add24h, trafficdata, and addons setups.

## Window Diagnostics

- [Window Diagnostics](window_diagnostics/summary.md): aggregate window-level diagnostics plus direct links to the exemplary best and worst windows.
- [Exemplary windows CSV](window_diagnostics/exemplary_windows.csv): ranked best and worst windows used to choose visual examples.
- [Row-level metrics CSV](window_diagnostics/row_level_metrics.csv): per-window metrics used for the diagnostics.
