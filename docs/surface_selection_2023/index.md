# Surface Selection 2023

This section collects the generated documentation artifacts for the METRAQ NOX 2023 surface-selection comparison.

The experiment set compares ten aligned sessions that vary only:

- ensemble use and size
- surface selection (`validation`, `validation_weighted`, `validation_median`, `last`)
- `k_best_n`

## Summary Pages

- [Performance summary](summary_paper_performance.md): aggregate means, win rates, Friedman mean ranks, Friedman omnibus tests, and pairwise Wilcoxon post-hoc comparisons with Holm adjustment for the ten surface-selection sessions.
- [Summary](interim_summary.md): narrative comparison of ensemble vs no ensemble and the `validation`, `validation_weighted`, `validation_median`, and `last` selectors.
